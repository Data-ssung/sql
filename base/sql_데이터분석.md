## 데이터분석을 위한 SQL 

### SQL 소개
- 데이터베이스와 통신을 하는 언어
- DQL(Data Query Language, 질의어)와 DML(Data Manipulation Language, 조작어) 구성

>
    1. 데이터 조회 : `SELECT`
    2. 데이터 생성 : `INSERT`
    3. 데이터 수정 : `UPDATE`
    4. 데이터 삭제 : `DELETE` 

### 데이터 분석을 위한 SQL
- 데이터 분석을 위해서 DB에서 조건에 맞는 데이터를 추출해서 가져오는 역량이 필요
- 1~4번은 엔지니어의 역량이지만, 잘 축적된 데이터를 가져오는 것은 데이터분석가의 역량
